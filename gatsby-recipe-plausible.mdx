# Setup Gatsby with Plausible Analytics

[Plausible](https://plausible.io/) is a simple and privacy-friendly alternative
to Google Analytics. It doesnâ€™t use cookies and is fully compliant with GDPR,
CCPA and PECR.

This recipe will help install and configure `@dnwjn/gatsby-plugin-plausible`.

---

Install the npm package:

<NPMPackage name="@dnwjn/gatsby-plugin-plausible" />

---

Add the plugin `@dnwjn/gatsby-plugin-plausible` to your `gatsby-config.js`.

<GatsbyPlugin
  name="@dnwjn/gatsby-plugin-plausible"
  options={{
    // Replace with the domain you configured in the Plausible dashboard
    domain: 'your-gatsby-site.tld',

    // If you are self-hosting Plausible:
    // plausibleDomain: 'your-plausible-instance.tld',

    // If you want to use a custom script:
    // plausibleScript: 'script.outbound-links.js',

    // If you want to exclude specific paths:
    // excludePaths: ['/exclude-path'],
  }}
/>

---

Read more about this plugin [here](https://www.github.com/dnwjn/gatsby-plugin-plausible).

Read more about Plausible [here](https://docs.plausible.io).
